Recommendations to Decouple the System

▼

9.1 Immediate Actions — Security (Do First)

  1. Rotate all exposed credentials (SendGrid, Twilio, Bitly, DB passwords)
  2. Move secrets to secure storage (Azure Key Vault / environment variables)
  3. Remove plain-text passwords from all config files in source control
  4. Add config files with secrets to .gitignore or use config transforms
